basis.js ‐ javascript framework for building client side web-applications.

At this moment, it's not just a framework, but also some tools for developers:

  • basisjs-tools ‐ CLI based on node.js contains builder, dev web server and more. Visit project page for more details.
  • chrome plugin ‐ a Google Chrome plugin, that maintains live editing app templates and translations.
